# 个人作品集网站 PRD

## 产品概述
- 个人作品集网站，用于展示个人项目、技能和联系方式。

## 核心功能
1. 首页：大标题 + 简介 + 头像
2. 关于我：详细介绍 + 技能列表
3. 项目展示：项目卡片列表，每个卡片包含项目名称、项目截图、简短描述、技术栈、项目链接
4. 联系方式：邮箱、GitHub、微信、社交媒体链接

## 设计要求
- 简洁现代的设计风格
- 流畅的滚动动画
- 响应式设计（PC、平板、移动端）
- 界面色调风格参考： https://www.pinterest.com/niu36910/electric-revolution/
- 动画参考： https://www.dylanbrouwer.design/
